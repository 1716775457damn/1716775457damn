## Featured Engineering Work

### 1. [rust-air](https://github.com/1716775457damn/rust-air)
**Cross-platform LAN file transfer built in pure Rust.**  
Includes CLI + desktop GUI, device discovery, file search, clipboard transfer, and sync workflow support.

### 2. [rust-seek](https://github.com/1716775457damn/rust-seek)
**High-performance local file and text search tool.**  
Native Rust GUI with filename/content search, regex support, GBK/UTF-8 handling, and fast large-directory traversal.

### 3. [visual-automation-designer](https://github.com/1716775457damn/visual-automation-designer)
**Visual desktop automation workflow builder.**  
A drag-and-drop automation designer using React Flow + Tauri + Rust, with flow editing, execution control, and desktop packaging.

### 4. [ESP-CSI-3D-Digital-Twin-Radar-System-](https://github.com/1716775457damn/ESP-CSI-3D-Digital-Twin-Radar-System-)
**Wi-Fi CSI sensing + 3D digital twin radar experiment.**  
Combines ESP32-S3, embedded Rust, DSP, and PyTorch-based sequence modeling for motion sensing and spatial visualization.

### 5. [K210_machanical-arm](https://github.com/1716775457damn/K210_machanical-arm)
**Computer vision robotic arm with patrol car integration.**  
Combines K210 vision inference, MCU control, IoT connectivity, and object-grabbing workflow logic.
